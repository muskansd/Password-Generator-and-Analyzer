PASSWORD TOOL

PasswordTool is a standalone Python application that allows you to analyze the strength of a password, generate strong passwords, and copy them to the clipboard. It features a real-time password strength meter, a view/hide password option, and a user-friendly GUI built with Tkinter.

FEATURES

Password Strength Analyzer

Shows password strength as Weak, Moderate, Strong, or Very Strong.

Real-time analysis as you type.

Color-coded strength bar for visual feedback.

Password Generator

Generate strong random passwords containing uppercase, lowercase, digits, and symbols.

Easily copy the generated password.

View/Hide Password

Toggle visibility of the password while typing.

Copy to Clipboard

Copy any password with a single click.

Standalone & Offline

Runs locally using Python and Tkinter. No internet required.

INSTALLATION

Ensure Python 3.x is installed.

Tkinter comes pre-installed with most Python distributions. If not, install it:

On Debian/Ubuntu: sudo apt-get install python3-tk

Download the PasswordTool folder containing app.py.

USAGE

Open terminal or command prompt.

Navigate to the folder containing app.py.

Run the application using: python app.py

The GUI window will open.

HOW TO USE

Enter Password: Type your password in the input box.

View Strength: The strength meter updates in real-time.

Generate Password: Click "Generate Password" to create a strong random password.

View/Hide Password: Use the toggle button to show or hide your password.

Copy Password: Click "Copy Password" to copy the current password to the clipboard.
